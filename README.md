## Descripción

<p>Diseñé y desarrollé un pequeño e-commerce que vende libros de Desarrollo Frontend.</p> 
<p>La aplicación web es 100% responsive. Se adapta perfectamente a cualquier dispositivo: escritorio, tablet y móvil.</p>
<p>Todas las imagenes del proyecto se alojan en Cloudinary, servicio de gestión de imágenes y videos basados ​​en la nube.</p>
<p>La aplicación se conecta al servicio de base de datos de Firebase para hacer manejo de stock, y también para mostrar el número de seguimiento de compra al usuario.</p>

## Tecnologías utilizadas

<p>• Vite + React Js</p>
<p>• Cloudinary</p>
<p>• Firebase</p>
